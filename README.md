# Round Robin CPU Scheduling Simulator

A simple simulation of the Round Robin scheduling algorithm used in operating systems.

## 📌 What is Round Robin?
Round Robin is a CPU scheduling algorithm where each process is assigned a fixed time quantum and executed in a cyclic order.

## 🎯 Features
- Round Robin scheduling simulation
- Time quantum-based execution
- Process queue visualization (if 없음 → 추가 추천)
- Simple input/output flow
